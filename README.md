# Onin Dev Toolbox

Onin Dev Toolbox plugin for Onin

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Pack

```bash
pnpm pack:plugin
```

`pnpm pack:plugin` creates a `plugin.zip` that includes:

- `manifest.json`
- `icon.svg`
- `dist/`

This template includes a sample settings schema registered from background.ts.
